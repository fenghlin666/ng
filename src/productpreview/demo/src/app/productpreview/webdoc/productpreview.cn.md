# Productpreview 商品预览

<div class="used-tiny">

Productpreview 用来展示商品的预览内容。&nbsp;&nbsp;

+ 商品预览组件，响应式尺寸。
+ 屏幕分辨率>=1600，尺寸450px。
+ 1600>屏幕分辨率>=1440，尺寸380px。
+ 1440>屏幕分辨率>=1280，分辨率>1280，尺寸300px。

```typescript
import { TiProductpreviewModule } from '@opentiny/ng';
```
